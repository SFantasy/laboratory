Handle data in the browsers easily.

## Usage

### localStorage

* setLocalStorage

```
D.setLocalStorage(key, value);
```

* getLocalStorage

```
D.getLocalStorage(key);
// return => Object
```

* delLocalStorage

```
D.delLocalStorage(key);
```
